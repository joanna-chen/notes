# HTML Notes

## Intro
* **HTML:** HyperText Markup Language
  * **markup language:** programming language used to make text do more than just sit on page
* **CSS:** Cascading Style Sheets
* **tags:** ```< >```


```HTML
<!DOCTYPE html>
<html>
  <head>
    <title>MyWebpage</title>
  </head>

  <body>
    <p>Hello world!</p>
  </body> 
</html>
```
